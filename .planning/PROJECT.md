# Kazamiyuuji

## What This Is

A personal portfolio website for kazamiyuuji — a multi-page site with maximum visual effects and Frutiger cyber aesthetic. It's the user's personal space on the internet, designed to showcase images and provide an immersive browsing experience.

## Core Value

An eye-catching, highly interactive portfolio site that works beautifully across all pages with zero cost — free hosting, free domain, free everything.

## User Stories

- As a visitor, I land on a stunning home page with animations and effects that immediately show this is a creative, polished site
- As a visitor, I can navigate between pages (Home, About, Gallery, etc.) with smooth transitions
- As a visitor, I experience hover effects, parallax, 3D elements, and animations on every page
- As the site owner, I have a professional-looking domain (kazamiyuuji) without paying anything
- As the site owner, I can easily swap placeholder images for real content later

## Core Features

### MVP (Phase 1)

- Multi-page portfolio site (Home, About, Gallery, Contact)
- Frutiger cyber design: glossy elements, transparency, water/nature motifs, organic shapes, blue/teal/white palette
- Maximum interactivity: hover effects, CSS animations, parallax scrolling, 3D transforms, scroll-triggered animations, cursor effects
- Responsive design (mobile + desktop)
- Smooth page transitions
- Free hosting via Vercel/Netlify/GitHub Pages
- Free subdomain (kazamiyuuji.vercel.app or similar)
- Placeholder images throughout

### Nice-to-Have (v2)

- Custom domain setup guide
- Blog section
- Dark/light mode toggle
- Music/audio integration
- Easter eggs and hidden interactions

## Out of Scope

- Paid hosting or domains
- Backend/database functionality
- User authentication
- E-commerce
- CMS integration
- Mobile app

## Tech Stack

- **Framework**: Next.js (React) — best for page routing and static export
- **Styling**: Tailwind CSS + custom CSS for Frutiger cyber effects
- **Animations**: Framer Motion + CSS animations + Three.js for 3D
- **Hosting**: Vercel (free tier, zero config for Next.js)
- **Domain**: Free subdomain via Vercel

## Success Criteria

- [ ] Site loads fast and looks good on desktop and mobile
- [ ] All pages navigate smoothly with transitions
- [ ] Hover effects, animations, and 3D elements work on every page
- [ ] Frutiger cyber aesthetic is consistent across the site
- [ ] Site is live and accessible via a public URL
- [ ] Zero cost to deploy and maintain
